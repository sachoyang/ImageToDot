ImageToDot
이미지를 도트 이미지로 변환시켜주는 exe파일


기능
Open Image - image파일 열기
Export PNG - png파일로 내보내기
Color Picker - 그리기 펜 색변경
Resolution Trans - 해상도 변경
Sharp Sampling - 부드럽게 도트변환 시에 해제
Palette - 기존 존재하는 팔레트 색상으로 자동 변경
Show Grid - grid 보여주기/숨기기
Enable Dithering - dithering 여부



*color picker로 그릴 수 있으나, 아직 ctrl+z 되돌리기는 미구현
